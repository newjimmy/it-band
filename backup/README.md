# IT Solutions & Project Management Website

A modern, responsive website for an IT company offering both technical solutions and project management services.

## Features

- Responsive design that works on all devices
- Modern and professional look
- Smooth scrolling navigation
- Contact form
- Services showcase
- About section
- Social media integration

## Deployment Instructions

This website is designed to be deployed on GitHub Pages. Follow these steps to deploy:

1. Create a new repository on GitHub
2. Push these files to your repository
3. Go to your repository settings
4. Navigate to the "Pages" section
5. Under "Source", select "main" branch and "/ (root)" folder
6. Click "Save"
7. Your site will be published at `https://[your-username].github.io/[repository-name]`

## Customization

To customize the website:

1. Update the content in `index.html`
2. Modify colors and styles in `styles.css`
3. Add your own images and update the paths
4. Update contact information and social media links
5. Add your company logo

## Development

To run the website locally:

1. Clone the repository
2. Open `index.html` in your web browser
